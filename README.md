# AtlyssMods

Consolidated repository of small to medium mods, experiments and other utilities for Atlyss.

Anything which isn't important or sizeable enough to warrant its own repository will end up in here.

For other Atlyss projects, check out:
- ...my full mods:
  - [ModAudio](https://github.com/Marioalexsan/AtlyssModAudio) - Audio pack mod with support for replacements, overlays, basic scripting and audio logging
  - [AtlyssDiscordRichPresence](https://github.com/Marioalexsan/AtlyssDiscordRichPresence) - Discord Rich Presence integration for Atlyss
  - [HostModeration](https://github.com/Marioalexsan/AtlyssHostModeration) - Fully server side moderation tools for Atlyss
  - [AutoSaver](https://github.com/Marioalexsan/AutoSaver) - Improves Atlyss with periodic saves and versioned backups to prevent save corruptions
- ...utilities that I created:
  - [AtlyssTemplateMod](github.com/Marioalexsan/AtlyssTemplateMod) - template to get started with modding Atlyss using BepInEx 5
  - [AtlyssGLTF](https://github.com/Marioalexsan/AtlyssGLTF) - Standalone compilation of glTFast
  - [AtlyssLua](https://github.com/Marioalexsan/AtlyssLua) - Lua-CSharp dependency based on [this fork](https://github.com/Marioalexsan/Lua-CSharp-Atlyss)
  - [AtlyssPerfectGuard](https://github.com/Marioalexsan/AtlyssPerfectGuard) - WIP exploit prevention for servers
